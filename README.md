
# Итоговое домашнее задание «Автоматизированный мониторинг и реагирование на угрозы»


Описание работы скрипта

Скрипт выполняет автоматизированный анализ сетевых угроз, используя два источника данных:

  - логи системы обнаружения вторжений Suricata

  - API сервиса VirusTotal

Этапы работы:

1. Скрипт загружает JSON-логи Suricata (alerts-only.json).

2. С помощью библиотеки pandas извлекаются IP-адреса источников трафика.
      
3. Для каждого IP выполняется запрос к API VirusTotal.

4. Анализируется статистика обнаружений вредоносной активности.

5. Если количество вредоносных детектов превышает заданный порог, скрипт выводит предупреждение и имитирует блокировку IP.

6. Результаты анализа сохраняются в отчётных файлах:

  - threat_report.csv

  - threat_report.json

7. Далее строится график наиболее опасных IP-адресов и сохраняется в файл threat_graph.png.

Используемые библиотеки: requests, pandas, matplotlib, json, seaborn
